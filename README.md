SauceDemo Automation Project 🚀

Este es un framework de automatización profesional desarrollado con **Playwright** y **JavaScript**. 

## ✨ Características
- **POM (Page Object Model):** Estructura organizada y mantenible.
- **Data-Driven Testing:** Pruebas dinámicas usando archivos JSON.
- **Seguridad:** Manejo de credenciales mediante variables de entorno (`.env`).
- **Reportes:** Generación automática de evidencias (videos y capturas).

## 🛠️ Instalación
1. Clonar el repositorio.
2. Ejecutar `npm install`.
3. Crear un archivo `.env` basado en la configuración del proyecto.

## 🧪 Ejecución de Tests
```bash
npx playwright test
